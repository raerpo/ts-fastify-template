# Fastify + TypeScript Teamplate

This is a basic template with the basic's you'll need when starting a new Fastify project built with TypeScript.

Also includes:

- Tests with jest.
- Linting.

## Default Commands

```sh
# Install deps
yarn

# Build into dist/
yarn build

# Run tests
yarn test

# Run
yarn start

# Run in dev mode (with reload on save)
yarn start:dev
```
